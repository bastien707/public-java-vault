- **Underlying Data Structure**: Red-Black tree
- **Advantages**: Maintains sorted order of keys
#### Complexity

- **Insertion**: O(log n)
- **Access (get)**: O(log n)
- **Update (put)**: O(log n)
- **Deletion**: O(log n)
- **Search (containsKey)**: O(log n)

#### Use Case

- Use `TreeMap` when you need to store key-value pairs and require keys to be in a sorted order.
